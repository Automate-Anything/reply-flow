Mobile Inbox: Message Thread Experience

When a user taps a conversation, how should the message thread work on mobile?

A

Full-Screen Takeover (WhatsApp-style)

Tapping a conversation pushes a full-screen message view. Back button returns to the list. Contact info via a top-bar tap. Bottom tab bar hides while in a thread.

Key Consideration: Contact Panel & Actions

Regardless of thread style, the contact panel (currently a right slide-over) and conversation actions (archive, assign, label, snooze) need mobile treatment too.

Option A — Full Screen

  • Maximum space for typing and reading
  • Users already know this pattern (WhatsApp)
  • Clean, focused experience
  • Keyboard won't compete with bottom tabs

Option B — Slide-Over

  • Quick-switch between conversations
  • Feels more "desktop-like"
  • Bottom tabs always accessible
  • But: less typing space, more complex gestures
